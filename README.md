# NTAM_homework

## Parsing and analysis
Загрузите сырые данные https://drive.usercontent.google.com/download?id=1r96vJA_6X2k3UAxD7irPPG1qtKDW8PNT&authuser=0. Напишите пайплайн обработки таблицы, применив паттерн проектирования "цепочка ответственности". На выходе должен быть .npy файл, готовый к дальнейшему анализу. Проект должен иметь интерфейс 
`python app path/to/hh.csv` который создаст рядом x_data.npy и y_data.npy

Проект должен соответствовать всем best practices и кодстайлу.

Запустить проект:
`python app.py hh.csv'
в папку с заданием надо поместить файл hh.csv
